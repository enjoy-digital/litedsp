# I/Q unpack

`LiteDSPIQUnpack` — `litedsp.stream.adapt` — category `stream`

latency: 0 samples · CSR: no · bypass: no

## Overview

Unpack one wide ``data`` word into ``ratio`` I/Q samples (inverse of :class:`LiteDSPIQPack`).

## Parameters

| Parameter | Default | Type | Description |
|---|---|---|---|
| `ratio` | `4` | int | Number of I/Q samples carried per input word (>= 1); the input ``data`` width is 2*data_width*ratio bits, first sample taken from the LSBs. Match the packer's ratio. |
| `data_width` | `16` | int | Sample width in bits (signed Qm.n; default Q1.15). |

## Ports

| Port | Direction | Layout |
|---|---|---|
| `sink` | sink | real |
| `source` | source | iq |
